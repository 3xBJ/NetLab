# NetLab

LabNet is a GitLab SDK for .Net. This project is aimed at people who need to
automate processes and report generation.

## Features
At the present moment one can:

 Search:
  - Issues;
  - Milestones;
  - Merge Requests;
  - Labels.
 
 Create:
  - Milestones.
